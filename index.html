Installer

sudosudo su Passer en super-utilisateur

apt-get install +nom du paquet Installer le paquet

apt-get update Mettre à jour les paquets disponibles dans les dépôts APT

apt-get upgrade Mettre à jour les paquets installés

apt-get install +nom du paquet=2.2-1 Installer la version 2.2-1 du paquet

apt-get remove +nom du paquet Désinstaller le paquet mais pas ses fichiers de configuration

apt-get purge +nom du paquet Supprimer le paquet et ses fichiers de configuration

Créer/Copier/Supprimer

mkdir +nom du répertoire Créer un nouveau répertoire (à l'endroit où l'on est)

touch +nom du fichier Créer un nouveau fichier (à l'endroit où l'on est)

cat +nom du fichier Afficher les détails du fichier

cp +nom du fichier Copier le fichier

mv +nom du fichier +redirection/ Déplacer ou renommer le fichier

rm +nom du fichier Supprimer le fichier

rm *.txt Supprimer tous les fichiers .txt

rm -rf + nom du fichier/ Supprimer tout le répertoire et son contenu

Afficher

ls Afficher le contenu du répertoire

ls -l Afficher les infos de manière détaillée

ls -a Afficher les fichiers cachés

ls -h Afficher la taille des fichiers

ls -r Tri inversé

ls -t Trier les fichiers par date (du plus récent au plus ancien)

ls -S Trier par taille décroissante

ls -lhS Afficher les informations fichiers, avec les tailles et triés du plus grand au plus petit

Se déplacer

cd +nom du répertoire ou cd ~ +nom du répertoire Accéder directement au répertoire de l'utilisateur

cd / Se déplacer à la racine du disque

cd +nom du répertoire/ Se déplacer dans le répertoire

cd .. Remonter dans le répertoire parent de là où l'on se trouve

cd - Remonter dans le répertoire précédent

pwd Renvoyer le chemin absolu du répertoire courant

atom . Ouvrir l'éditeur de texte atom
